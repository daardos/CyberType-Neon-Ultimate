# ⌨️ NeonType: Cyberpunk Speed Typing Trainer 🚀

**NeonType** — это стильный и минималистичный тренажер слепой печати, созданный на чистом JavaScript. Проект ориентирован на точность (Accuracy) и скорость (WPM), упакованный в эстетику темного киберпанка.

---

## ⚡ Ключевой функционал (JS Engine)

В основе тренажера лежат продвинутые алгоритмы обработки текстовых потоков:

*   **🔍 Реальный посимвольный анализ:** Текст разбивается на массив DOM-узлов (`<span>`), что позволяет подсвечивать каждый символ отдельно (Correct/Error) в реальном времени.
*   **📈 Динамический расчет WPM:** Формула `(Characters / 5) / (Time)` пересчитывает твой результат каждую секунду, давая мгновенную обратную связь.
*   **🎧 Web Audio Engine:** Программная генерация звука "механических клавиш" через **Web Audio API** (без использования тяжелых MP3-файлов). Разная тональность для правильных нажатий и ошибок.
*   **🌍 Мультиязычный движок:** Поддержка RU/EN раскладок с мгновенной сменой словарей и алгоритмов проверки.
*   **💾 Local Storage Records:** Система сохранения личных рекордов (TOP-5) в локальном хранилище браузера. Твои достижения не пропадут после перезагрузки.
*   **🫨 Visual Feedback:** Эффект "тряски экрана" (Screen Shake) при совершении ошибки для усиления тактильного ощущения.

---

## 🛠 Технический стек

*   **Frontend:** HTML5 / CSS3 (Flexbox, CSS Variables, Animations).
*   **Core:** Vanilla JavaScript (ES6+).
*   **Audio:** Web Audio API (Oscillators & Gain Nodes).
*   **Storage:** Window.localStorage API.

---

## 🎮 Как пользоваться

1. Откройте `index.html`.
2. Выберите язык (RU/EN).
3. Просто начните печатать. Таймер запустится автоматически при первом нажатии.
4. Дойдите до конца текста, чтобы сохранить свой рекорд в таблицу.

---

## 🚧 Технические особенности реализации

*   **Hidden Input Focus:** Реализована логика "невидимого фокуса" — вы печатаете в скрытый `input`, а JS транслирует это на красиво оформленные символы.
*   **Adaptive Cursor:** Каретка (курсор) вычисляет координаты `offsetLeft` и `offsetTop` каждого символа, плавно следуя за твоим прогрессом.
*   **Performance:** Нулевая зависимость от внешних библиотек. Мгновенная загрузка и работа.
